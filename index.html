Google Maps Polyline Utility

  • DOCUMENTAÇÃO
Instruções:

 1. Insira a morada central do poligno no campo "Center map at" e depois clicar na tecla "SEARCH".
2. Clique no mapa para colocar o primeiro local do poligno.(Pode arrastar o marcador para ajustar a localização.)
3. Se necessário, especifique o nível apropriado para a localização no campo "Display Level".
4. Clique no botão "ADD LOCATION" para o adicionar à Lista de locais (Location list).
5. Repita para cada local que queira adicionar ao poligno. (Pode arrastar o marcador para ajustar a localização.)
6. A codificação do poligno aparecerá nos campos "Encinheled Polyline" e "Encoded Levels".
7. Copie o campo "Encinheled Polyline" e cole-o onde o pretende usar.

Este utilitário usa a API JavaScript do Google Maps.
